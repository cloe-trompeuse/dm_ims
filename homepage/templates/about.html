IMS
DM Inventory Management System


| About |


DM IMS is a simple inventory management web application created for the basic stock management requirements of a company. The software is capable of keeping track of incoming and outgoing stock with dynamically rendered bills. Each supplier is provided with a profile where they can see the history of their transactions with the company.


| Features |


  • Interactive Chart to display current available stock
  • Stock maintenance
  • Incoming transactions done through purchases
  • Outgoing transactions done through sales
  • Record of all transactions performed
  • Dynamically rendered bills for each transaction
  • Maintenance of separate profiles for suppliers
